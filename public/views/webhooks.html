Cara Kerja Webhook Pesan Masuk

Panel ini menjelaskan kapan webhook incoming message dikirim dan apa yang harus dicek.

Event
message
Pastikan saat add webhook, event message dicentang. Kalau tidak dicentang, pesan masuk tidak akan dikirim ke URL webhook.
Kapan Dikirim
1. Device menerima pesan WhatsApp masuk
2. Gateway simpan ke message_logs
3. Gateway kirim POST ke URL webhook
4. Event yang dikirim adalah message
Cara Cek
- Kirim pesan dari nomor lain ke device target
- Buka menu Logs atau Webhooks
- Cari event message
- Cek HTTP code, URL, dan payload di detail log
Contoh Payload Incoming
{
  "event": "message",
  "device_id": "device-anda",
  "timestamp": "2026-03-30T13:00:00.000Z",
  "data": {
    "id": "WA_MESSAGE_ID",
    "device_id": "device-anda",
    "from": "628xxxx@c.us",
    "from_num": "628xxxx@c.us",
    "to": "628yyyy@c.us",
    "to_num": "628yyyy@c.us",
    "body": "isi pesan masuk",
    "type": "chat",
    "is_read": 0
  }
}
